# EMPOWER - Premium Luxury Car Dealership Website

Welcome to the **EMPOWER** repository! This is a state-of-the-art, high-end luxury car dealership web application built to deliver a cinematic, performant, and deeply immersive digital experience.

## Overview
EMPOWER redefines the digital showroom. Designed with an emphasis on high-end aesthetics, performant typography, and dynamic animations, the application showcases luxury vehicles with smooth interactions.

## Key Features
- **Cinematic Experience**: Scroll-triggered animations and fluid transitions for a premium feel.
- **Performant Typography**: Sophisticated use of the Oswald font for a modern and sleek interface.
- **Interactive Showroom**: Full-screen interactive car configuration pages and high-quality viewing.
- **Smooth Scrolling**: Integrated Lenis smooth scroll for a seamless navigation experience.
- **Responsive Design**: Flawlessly adapts to any viewport—from mobile devices to ultra-wide desktop monitors.

## Tech Stack
- **Framework**: [Next.js 16](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scrolling**: [Lenis](https://github.com/studio-freight/lenis)
- **3D / Interactions**: [Three.js](https://threejs.org/) & React Three Fiber

## Getting Started

### Prerequisites
Make sure you have Node.js (v18+ recommended) installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/car_website.git
   cd car_website
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to explore the showroom.

## Project Structure
- `src/components/`: Reusable, styled UI and interactive components (e.g., VideoSection, FAQ, Features, Footer).
- `src/app/`: Next.js App Router pages and layouts.
